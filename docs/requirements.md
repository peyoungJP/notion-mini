# Notion-mini (Daily Notes) Requirements

## Goal
Personal notes app with login and notes CRUD.

## MVP
- Login/logout
- List notes (only own notes)
- Create / Edit / Delete note
- Note fields: title, body, created_at, updated_at

## Acceptance Criteria (MVP)
- User can sign in and sign out
- Signed-in user sees only their notes
- Create note -> appears in list immediately
- Edit note -> changes are persisted
- Delete note -> disappears from list
